[Hivedscan](https://github.com/thisonedev/vault/tree/master/hive/hivedscan) is an all-in-one blockchain explorer for Hive-based dapps.

The project contains the following components:

1. Hive DAO - a decentralized budgeting and governance system
2. Hive Engine - a platform for creating tokens via smart contracts

![01](../assets/hivedscan.png)

## Setup

### 1. Clone the repository

```
https://github.com/thisonedev/vault.git
```

### 2. Enter hive/hivedscan directory

```
cd hive/hivedscan
```

### 3. Install dependencies

```
npm install
```

## Running a project

```bash
npm start
```

## Building a project

```bash
npm build
```
